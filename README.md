# Books Catalog (Yii1) Инфотек

- Книги (название, год, описание, ISBN, обложка)
- Авторы (ФИО), связь many-to-many с книгами
- Регистрация/авторизация
- Гость: просмотр + подписка на автора (телефон)
- Юзер: CRUD книг и авторов
- Отчет ТОП-10 авторов по году
- SMS уведомления через smspilot.ru (тестовый ключ)
